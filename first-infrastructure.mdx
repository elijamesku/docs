## Quick Example

Generate a simple web server:

1. Open Kyrna
2. Click "New Project"
3. Type: "Create a web server on AWS"
4. Press Cmd+Enter
5. Review the generated code
6. Click "Apply" to deploy

See the full [Quickstart Guide](/quickstart) for details.
