Welcome to Kyrna Docs. Kyrna is an AI-powered platform for infrastructure code generation with built-in cost tracking and security scanning.

## What is Kyrna?

Kyrna helps you write infrastructure code faster using AI. Type what you want in natural language, and Kyrna generates production-ready Terraform code.

## Current Features

What's implemented and working:

### AI Code Generation

- Natural language to Terraform conversion
- Claude 4.5 Sonnet or GPT-5 support
- Context-aware generation
- Multi-resource support (VPC, EC2, RDS, S3, etc.)

### Cost Intelligence

- Real-time AWS cost estimation
- 15+ AWS resource types supported
- Per-resource cost breakdown
- Budget alerts

### Security & Compliance

- Drift detection (code vs deployed)
- Basic security scanning
- Terraform validate/fmt integration
- Auto-heal for common issues

### Development Tools

- Diff approval workflow
- Git integration (commit, branch, PR)
- Architecture diagram generation
- Documentation generation

### Supported Platform

- **Cloud Provider:** AWS (only)
- **IaC Language:** Terraform (only)
- **AI Models:** Claude 4.5 Sonnet, GPT-5
- more soon...
