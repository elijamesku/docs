## Essential Shortcuts

Speed up your workflow with Kyrna's keyboard shortcuts.

---

## AI & Agent Mode

### Agent Mode

<kbd>Ctrl</kbd>

 + <kbd>Shift</kbd>

 + <kbd>L</kbd>

Toggle between Ask mode (questions) and Agent mode (execution).

**Use when:**

- Switching from exploration to implementation
- Ready to generate actual infrastructure code

---

## UI Navigation

### Show/Hide Terminal

<kbd>Ctrl</kbd>

 + <kbd>J</kbd>

Toggle the terminal panel at the bottom.

**Use when:**

- Running Terraform commands
- Checking git status
- Executing local commands

---

### Hide Files (Toggle Sidebar)

<kbd>Ctrl</kbd>

 + <kbd>B</kbd>

Show or hide the file explorer sidebar.

**Use when:**

- Need more screen space for code
- Want to focus on current file
- Maximizing editor view

---

### Search Files

<kbd>Ctrl</kbd>

 + <kbd>P</kbd>

Quick file search - type to find and open files instantly.

**Use when:**

- Navigating large projects
- Quickly jumping between files
- Can't remember exact file location

### Chat

<kbd>Ctrl</kbd>

 + <kbd>K</kbd>

Move cursor to chat input field.

**Use when:**

- Want to ask AI a question quickly
- Done editing, ready to chat

---

### Send Message

<kbd>Enter</kbd>

Send message in chat (when focused).

---

### New Line in Chat

<kbd>Shift</kbd>

 + <kbd>Enter</kbd>

Add new line in chat message without sending.

---

## Quick Actions

### Quick Command Palette

<kbd>Ctrl</kbd>

 + <kbd>Shift</kbd>

 + <kbd>P</kbd>

 (Windows/Linux)\
<kbd>Cmd</kbd>

 + <kbd>Shift</kbd>

 + <kbd>P</kbd>

 (macOS)

Access all available commands.

---

### Copy Path

<kbd>Ctrl</kbd>

 + <kbd>K</kbd>

 then <kbd>P</kbd>

Copy current file's path to clipboard.

---

### Duplicate Line

<kbd>Shift</kbd>

 + <kbd>Alt</kbd>

 + <kbd>↓</kbd>

 (Windows/Linux)\
<kbd>Shift</kbd>

 + <kbd>Option</kbd>

 + <kbd>↓</kbd>

 (macOS)

Duplicate current line below.

---

### Move Line Up/Down

<kbd>Alt</kbd>

 + <kbd>↑</kbd>

 / <kbd>↓</kbd>

 (Windows/Linux)\
<kbd>Option</kbd>

 + <kbd>↑</kbd>

 / <kbd>↓</kbd>

 (macOS)

Move current line up or down.

---

### Delete Line

<kbd>Ctrl</kbd>

 + <kbd>Shift</kbd>

 + <kbd>K</kbd>

 (Windows/Linux)\
<kbd>Cmd</kbd>

 + <kbd>Shift</kbd>

 + <kbd>K</kbd>

 (macOS)

Delete entire line.

---

## Multi-Cursor Editing

### Add Cursor Above/Below

<kbd>Ctrl</kbd>

 + <kbd>Alt</kbd>

 + <kbd>↑</kbd>

 / <kbd>↓</kbd>

 (Windows/Linux)\
<kbd>Cmd</kbd>

 + <kbd>Option</kbd>

 + <kbd>↑</kbd>

 / <kbd>↓</kbd>

 (macOS)

Add cursor above or below current position.

---

### Add Cursor to All Occurrences

<kbd>Ctrl</kbd>

 + <kbd>Shift</kbd>

 + <kbd>L</kbd>

 (Windows/Linux)\
<kbd>Cmd</kbd>

 + <kbd>Shift</kbd>

 + <kbd>L</kbd>

 (macOS)

Select all occurrences of current selection.

---

### Add Next Occurrence

<kbd>Ctrl</kbd>

 + <kbd>D</kbd>

 (Windows/Linux)\
<kbd>Cmd</kbd>

 + <kbd>D</kbd>

 (macOS)

Add next occurrence of selection to cursors.

## Pro Tips

### Combine Shortcuts

**Quick edit workflow:**

1. <kbd>Ctrl</kbd>

    + <kbd>P</kbd>

    - Find file
2. <kbd>Ctrl</kbd>

    + <kbd>F</kbd>

    - Find text
3. <kbd>Ctrl</kbd>

    + <kbd>S</kbd>

    - Save
4. <kbd>Ctrl</kbd>

    + <kbd>K</kbd>

    - Ask AI about changes

**Agent mode workflow:**

1. <kbd>Ctrl</kbd>

    + <kbd>Shift</kbd>

    + <kbd>L</kbd>

    - Enable agent mode
2. Type your infrastructure request
3. <kbd>Enter</kbd>

    - Send
4. Review proposals
5. <kbd>Ctrl</kbd>

    + <kbd>S</kbd>

    - Save accepted files
