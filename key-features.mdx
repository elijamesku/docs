## Features 

This is what you get today (MVP):

### AI Code Generation

**What works:**

- Claude 4.5 Sonnet or GPT-5
- Natural language \>\> Terraform
- Multi-resource generation
- AWS services (EC2, RDS, S3, VPC, Lambda, etc.)

**Status:** Working

### Cost Tracking

Real-time AWS cost estimates while you code.

**What works:**

- 15+ AWS resource types
- Monthly/hourly costs
- Per-resource breakdown
- Budget alerts

**Limitations:**

- AWS only (no GCP/Azure)
- Estimates only (not actual bills)

**Status:** Working
