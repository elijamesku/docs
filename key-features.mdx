## Features

This is what you get today (MVP):

### AI Code Generation

Generate Terraform from natural language.

**What works:**

- Claude Opus 4.5  or GPT-5
- Natural language \>\> Terraform
- Multi-resource generation
- AWS services (30+ resource types)
- DigitalOcean services (18+ resource types)

**Status:** Working

### Cost Tracking

Real-time cost estimates while you code.

**What works:**

- 30+ AWS resource types
- 18+ DigitalOcean resource types
- Monthly/hourly costs
- Per-resource breakdown
- Budget alerts

**Limitations:**

- AWS and DigitalOcean only (GCP/Azure coming soon)
- Estimates only (not actual bills)

**Status:** Working
