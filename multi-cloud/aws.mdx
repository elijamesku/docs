## AWS Support

Kyrna currently supports AWS only.

**What works:**
- EC2, RDS, DynamoDB, Lambda
- S3, EBS, CloudWatch
- VPC, Subnets, Security Groups
- ELB, ALB, Route53
- And more...

**Cost tracking:** 15+ AWS resource types

**Status:** Working

## Other Clouds

GCP, Azure, and DigitalOcean support is planned but not built yet.

**Current:** AWS only
