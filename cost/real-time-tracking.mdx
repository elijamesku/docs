## How It Works

Kyrna tracks AWS costs in real-time while you write Terraform code.

**Status:** Working for AWS only

## What You See

<CardGroup cols={2}>
  <Card title="Per-Resource Cost" icon="tag">
    Each AWS resource shows its cost
  </Card>
  <Card title="Total Cost" icon="calculator">
    Sum of all resources in your project
  </Card>
</CardGroup>

## Example

```hcl
resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t3.micro"          # üí∞ $8.35/month
}

resource "aws_db_instance" "main" {
  instance_class   = "db.t3.medium"   # üí∞ $61.32/month
  engine          = "postgres"
  allocated_storage = 100              # üí∞ $11.50/month
}

# Total: $81.17/month
```

## Current Limitations

<Warning>
  **AWS Only**

  Currently only supports AWS. GCP, Azure, and DigitalOcean support coming later.
</Warning>

**What's NOT included:**

- Data transfer costs (highly variable)
- API request costs
- Reserved Instance discounts
- Actual billing (estimates only)
