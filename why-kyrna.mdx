## The Problem

Writing and managing infrastructure code is slow:

- Hours reading Terraform docs
- Trial and error with syntax
- Cost surprises after deployment
- Security issues go unnoticed

## Our Solution (Current State)

### What Works Now

**AI Code Generation:**

- Natural language \>\> Terraform
- AWS resources
- Diff editor
- Auto heal logic

**Cost Tracking:**

- Real-time AWS estimates
- 15+ resource types
- Budget alerts

**Security:**

- Basic drift detection
- Terraform validation
- Auto-heal for errors

**Status:** MVP, improving

## Who Should Use Kyrna

### Good Fit

- **Learning Terraform** - Generate examples to learn from
- **AWS/DigitalOcean Users** - We support both providers
- **Individual Developers** - Full feature set available
- **Teams** - Real-time collaboration and team management
- **Prototyping** - Quick infrastructure setup
- **Cost-Conscious** - See costs before deploying
- **Migrating Infrastructure** - Import existing AWS infrastructure

### Coming soon...

- **Additional cloud providers** - GCP, Azure support
- **Large enterprises** - SSO/RBAC (in progress)
- **Non-Terraform users** - Terraform only
