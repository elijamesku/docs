Welcome to Kyrna Docs. Kyrna is an AI-powered platform for infrastructure code generation with built-in cost tracking, security scanning, and **real-time team collaboration**.

## What is Kyrna?

Kyrna helps you write infrastructure code faster using AI. Type what you want in natural language, and Kyrna generates production-ready Terraform code. Collaborate with your team in real-time, manage infrastructure across AWS and DigitalOcean, and import existing infrastructure.

## Current Features

What's implemented and working:

### AI Code Generation

- Natural language to Terraform conversion
- Claude Opus 4.5 or GPT-5 support
- Context-aware generation
- Multi-resource support (VPC, EC2, RDS, S3, etc.)

### Cost Intelligence

- Real-time cost estimation for AWS and DigitalOcean
- 30+ AWS resource types supported
- 18+ DigitalOcean resource types supported
- Per-resource cost breakdown
- Budget alerts

### Security & Compliance

- Drift detection (code vs deployed)
- Basic security scanning
- Terraform validate/fmt integration
- Auto-heal for common issues

### Development Tools

- Diff approval workflow
- Git integration (commit, branch, PR)
- Architecture diagram generation
- Documentation generation

### Team Collaboration

- Real-time presence tracking
- Live file activity notifications
- Team PR workflow (collaborative pull requests)
- Conflict prevention

### Team Management

- Create teams and invite members
- Role-based access control (Admin, Developer, Viewer)
- Shared repositories
- Team billing and subscriptions

### Supported Platforms

- **Cloud Providers:** AWS (30+ resources), DigitalOcean (18+ resources)
- **IaC Language:** Terraform
- **AI Models:** Claude 4.5 Sonnet, GPT-5
- **Import Formats:** AWS CLI, CloudFormation, AWS Config, Natural Language
- more soon...
